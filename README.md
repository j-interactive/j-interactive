# Joe Griffiths — Personal Profile

A minimal personal profile page deployed at [j-interactive.dev](https://j-interactive.dev) via GitHub Pages.

## Stack

- Vanilla HTML and CSS, no frameworks or build tools
- Google Fonts: [Geist](https://fonts.google.com/specimen/Geist)

## Structure

```
├── index.html          # All HTML and CSS in one file
├── Joseph-Griffiths-CV.pdf
├── favicon.ico
└── CNAME               # Maps to j-interactive.dev
```

## Usage

Open `index.html` directly in a browser — no build step required.
